# Jansori 잔소리

Your caring desk companion — posture, walk, and hydration reminders.

잔소리 means "caring nagging" in Korean — like a parent telling you to sit up straight. Built by [BurgundyTable](http://burgundytable.com).

## Features

- **Three reminder types:** Posture (🪑), Walk breaks (🚶), Hydration (💧)
- **Independent intervals** for each type (5–120 minutes)
- **4 built-in sounds:** Gentle Bell, Singing Bowl, Wood Knock, Soft Chime
- **Custom sound upload** — use any audio file under 1 MB
- **Bilingual** — switch between English and Korean with one click
- **Desktop notifications** with witty, rotating messages
- **Dark burgundy UI** — warm, not corporate
- **Zero data collection** — all settings stored locally on your device

### Developer (Load Unpacked)

1. Clone or download this repository
2. **Chrome:** Go to `chrome://extensions` → Enable "Developer mode" → Click "Load unpacked" → Select the project folder
3. **Firefox:** Go to `about:debugging` → "This Firefox" → "Load Temporary Add-on" → Select `manifest.json`

## How It Works

Click the extension icon to open the popup. Toggle reminders on/off, set intervals, choose a sound, and adjust volume. Jansori runs in the background and sends you gentle nudges at your chosen intervals.

## Privacy

Jansori stores all settings locally on your device. No data is collected, transmitted, or shared. See [PRIVACY.md](PRIVACY.md) for the full policy.
